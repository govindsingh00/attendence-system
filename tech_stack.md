# Tech Stack Justification

## Backend: Django + Django REST Framework
- Django is fixed choice by team
- DRF handles REST APIs easily
- Built-in admin panel for free
- Best Python web framework

## Database: PostgreSQL
- Best integration with Django
- Handles relational data perfectly
- Free and open source
- Better than SQLite for multi-user system
- Supports complex queries for reports

## Frontend: React.js
- Works perfectly with Django REST APIs
- Handles real-time UI updates
- Large community and support
- Component based architecture

## ML Module: Python + OpenCV
- Face recognition using OpenCV library
- Auto marks attendance via webcam
- Posts matched user data to Django API
- Confidence score stored in FACE_LOG

## Version Control: Git + GitHub
- Feature branches for each member
- Pull requests for code review
- Everyone works independently
